# UrbanScanIA: Infrastructure Intelligence Platform

UrbanScanIA is a high-performance platform designed for the automated analysis and classification of urban and natural satellite imagery. Utilizing state-of-the-art Deep Learning models, the system provides accurate identification of infrastructure patterns to assist in urban planning and environmental monitoring.

![UrbanScanIA Interface](file:///c:/Users/CARLOS/Documents/GITHUB/UrbanScanIA/frontend/public/images/hero.png)

## Project Overview

The architecture follows a decoupled micro-service approach, separating the computational heavy-lifting of the Artificial Intelligence models from the high-reactivity user interface.

### Key Capabilities
- Automated classification of land use using the EuroSAT dataset benchmark.
- High-accuracy inference leveraging ResNet50 neural networks.
- Real-time multispectral infrastructure analysis.
- Modern, responsive interface optimized for professional GIS workflows.

## Technical Specifications

### Backend (Core Engine)
- **Framework**: FastAPI (Asynchronous Python Framework)
- **Deep Learning Core**: PyTorch
- **Model Architecture**: ResNet50 (Pre-trained on EuroSAT)
- **Image Processing**: Pillow and HuggingFace Transformers
- **Server**: Uvicorn (ASGI implementation)

### Frontend (User Interface)
- **Framework**: Astro (Static Site Generator & Component Islands)
- **Design System**: Custom CSS with Glassmorphism specifications
- **Typography**: Google Fonts (Outfit)
- **Performance**: Optimized zero-JS initial load with reactive client-side fragments

## Deployment and Installation

### Backend Environment Setup
1. Change directory to the backend module:
   ```bash
   cd backend
   ```
2. Initialize a Python virtual environment:
   ```bash
   python -m venv venv
   ```
3. Activate the virtual environment:
   - On Windows: `.\venv\Scripts\activate`
   - On Linux/macOS: `source venv/bin/activate`
4. Install the required libraries:
   ```bash
   pip install -r requirements.txt
   ```
5. Execute the API server:
   ```bash
   uvicorn main:app --reload
   ```

### Frontend Environment Setup
1. Change directory to the frontend module:
   ```bash
   cd frontend
   ```
2. Install the Node.js dependencies:
   ```bash
   npm install
   ```
3. Initialize the development environment:
   ```bash
   npm run dev
   ```
4. Access the platform at [http://localhost:4321](http://localhost:4321)

## API Documentation
The API documentation is automatically generated by FastAPI and can be accessed at:
- Swagger UI: `http://localhost:8000/docs`
- ReDoc: `http://localhost:8000/redoc`

## Data Provenance
The underlying model is trained on the EuroSAT dataset, which consists of Sentinel-2 satellite images covering 13 spectral bands and 10 classes, including industrial buildings, highways, and residential areas.
